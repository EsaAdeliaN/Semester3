Pengenalan Tentang Rekayasa Perangkat Lunak

Program Studi Rekayasa Perangkat Lunak (RPL) merupakan domain Bidang Ilmu Informatika & Komputer, dalam aspek pengembangan software mulai dari tahap awal spesifikasi, desain, konstruksi, testing sampai pemeliharaan.

Profesi-profesi lulusan RPL antara lain Programmer, System Analyst, Software Quality Assurance Engineer (SQAE), Database Administrator, Software Architect, Software Support, Konsultan IT, Web Designer.

Di indonesia, Program Studi RPL Kurang dari 10. Politeknik Balekambang merupakan satu-satunya Perguruan Tinggi Swasta yang memiliki program studi Rekayasa Perangkat Lunak (Software Enginering). Selain itu, program studi ini mencakup seluruh bidang domain komputer lain.


Visi & Misi
Rekayasa Perangkat Lunak

  • Visi

    “Menjadi Program Studi Vokasi di bidang Rekayasa Perangkat Lunak Berdaya Saing Global, Profesional dan berjiwa Technopreunership.”

  • MISI

    1. Menyelenggarakan pendidikan vokasi di bidang Teknologi Rekayasa Perangkat Lunak sesuai dengan perkembangan ilmu pengetahuan dan teknologi yang berjiwa Technopreunership.

    2. Menyelenggarakan riset-riset di bidang Teknologi Rekayasa Perangkat Lunak yang inovatif, berkualitas dan bermanfaat untuk pengembangan ilmu pengetahuan dan teknologi.

    3. Meningkatkan kerjasama strategis yang produktif dengan lembaga pendidikan, pemerintahan dan dunia usaha ditingkat nasional dan Internasional dalam mengatasi permasalahan di bidang Teknologi Rekayasa Perangkat Lunak


KOMPETENSI LULUSAN
REKAYASA PERANGKAT LUNAK
POLITEKNIK BALEKAMBANG


• Senior Programmer

Memimpin produksi pada suatu proyek pembuatan software sekaligus memastikan standar kualitas software yang dibuat dengan cara mengelola, membuat jadwal dan membagi setiap tugas kepada programmer sesuai dengan job diskripsi yang sudah ditentukan sebelumnya

• Software Developer
Membangun suatu software sesuai dengan konsep Software Development Life Cycle (SDLC)sesuai dengan kebutuhan klien untuk kebutuhan software komputer atau perangkat lain

• Web Developer
Merancang dan membuat suatu website sesuai dengan standar pembuatan website sekaligus bertanggung jawab dalam perawatan dan pemeliharaan.

• Game Developer
Merancang, membuat dan melakukan pemeliharaan suatu software game yang interaktif dengan menggabungkan teks, grafik, animasi, pencitraan, audio-video display, dan media lainnya.

• Database Administrator (DBA)
Melakukan penyimpanan dan mengorganisasikan data dengan menggunakan perangkat lunak khusus serta memastikan bahwa data yang tersedia aman dan terlindungi dari akses ilegal.
